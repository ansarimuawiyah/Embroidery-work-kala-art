# Embroidery-work-kala-art

## Kala Art - आपकी रचनात्मक दुनिया

A beautiful website showcasing handmade embroidery and art work (कढ़ाई और कलाकृतियां).

### Features
- 🎨 Responsive design
- 📱 Mobile-friendly navigation
- 🖼️ Interactive gallery with filtering
- 📧 Contact form
- 🌍 Hindi and English support
- ✨ Smooth animations and transitions

### Sections
1. **Home/Hero** - Eye-catching landing section
2. **About** - Information about the business with statistics
3. **Gallery** - Showcase of artworks with category filtering (Embroidery, Painting, Sculpture)
4. **Services** - Key services offered
5. **Contact** - Contact information and message form

### Technologies Used
- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (ES6+)
- Font Awesome Icons

### How to Use
1. Clone the repository
2. Open `index.html` in your browser
3. Navigate through different sections
4. Use the gallery filters to view different types of artwork
5. Fill out the contact form to get in touch

### Responsive Design
The website is fully responsive and works seamlessly on:
- Desktop
- Tablet
- Mobile devices

### Color Scheme
- Primary Color: #8B4513 (Saddle Brown)
- Secondary Color: #D2691E (Chocolate)
- Accent Color: #FF6B6B (Light Red)
