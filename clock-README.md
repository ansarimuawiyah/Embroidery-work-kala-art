# Digital Clock - Multiple Time Zones

A modern, interactive digital clock application that displays the current time across multiple time zones simultaneously.

## Features

✨ **Key Features:**
- 🕐 Real-time digital clock display with seconds
- 🌍 Support for 30+ major world time zones
- 🎨 Both 24-hour and 12-hour (AM/PM) format
- 🌙 Dark mode and light mode themes
- ⏰ Analog clock representation for each timezone
- 🔄 Smooth animations and transitions
- 💾 Persistent storage (LocalStorage)
- 📱 Fully responsive design
- ✏️ Add/remove time zones dynamically
- 🔍 Timezone search functionality

## Supported Timezones

- Asia: Kolkata, Tokyo, Shanghai, Dubai, Hong Kong, Singapore, Bangkok, Jakarta, Manila, Istanbul, Seoul
- Americas: New York, Los Angeles, Chicago, Denver, Phoenix, Toronto, Mexico City, Sao Paulo
- Europe: London, Paris, Berlin, Moscow, Athens
- Africa: Cairo, Johannesburg
- Pacific: Sydney, Auckland, Fiji
- Indian: Mauritius

## How to Use

1. **Open** `clock.html` in your web browser
2. **View** the default time zones (India, US, UK, Japan)
3. **Add** new time zones by clicking "+ Add Time Zone" button
4. **Search** for specific timezones in the modal
5. **Remove** any timezone by clicking the trash icon
6. **Switch** between 24-hour and 12-hour formats
7. **Toggle** dark/light mode for comfortable viewing
